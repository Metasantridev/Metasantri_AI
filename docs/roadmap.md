# Roadmap Metasantri AI

> Status: 🧪 Prototype — rilis publik direncanakan segera

## ✅ Sudah Dibangun (Prototype)

- [x] Chat AI 5 mode (Flash / Kang Santri / Ustadz / Kiyai / Ulama)
- [x] RAG Kitab Pesantren (basis awal, terus diperluas)
- [x] Sistem akses bertingkat (Free / Pro / Plus)
- [x] Admin dashboard
- [x] Bahtsul Masail
- [x] Hafalan santri + Perpustakaan kitab

- [x] Landing page dengan real-time sky background
- [x] Bisa diinstall ke HP (PWA)
- [x] I'rab nahwu otomatis dengan istilah Arab asli
- [x] Login Google + Email

## 🚧 Sedang Dikerjakan

- [ ] Perluasan basis kitab menuju target **6.700+ kitab**
- [ ] Integrasi pembayaran untuk upgrade akses
- [ ] Penyempurnaan kualitas rujukan (RAG)

## 🔮 Rencana Ke Depan

- [ ] **Fitur Sosial** — leaderboard hafalan, share jawaban, forum tanya-jawab santri
- [ ] **Mobile App Native** — notifikasi pengingat sholat & hafalan
- [ ] **Integrasi Pesantren** — kerja sama untuk pesantren yang ingin mengadopsi sistem serupa
- [ ] **Konten Multimedia** — audio tilawah per ayat, video penjelasan ulama

## 💡 Ide Jangka Panjang

- Metasantri untuk anak-anak (konten disesuaikan usia)
- Integrasi dengan kitab digital pesantren (upload mandiri)
- API publik untuk developer muslim Indonesia
